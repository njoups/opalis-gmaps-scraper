# Opalis Gmaps Scraper

Scraper sederhana untuk mengambil data bisnis dari Google Maps menggunakan Puppeteer.

## Cara Pakai

1. Install dependensi:

```bash
npm install
```

2. Jalankan:

```bash
node index.js
```

Data akan tersimpan di `results.csv`.

## Konfigurasi

Edit keyword & lokasi di file `index.js`:

```js
const keyword = 'barber';
const location = 'Los Angeles';
```
